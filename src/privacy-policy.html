This Privacy Policy describes how your information is handled in the private instance of n8n hosted at n8n.fedrian.dev.

1. Overview

This application is a private, personal automation instance. It is used exclusively by the domain owner for workflow automation and personal productivity.

2. Use of Google User Data

If you authorize this application to access your Google account via OAuth, the application will only access the specific data (scopes) you have explicitly permitted. This data is used solely to execute the automations you configure within the private instance (e.g., synchronizing emails, updating spreadsheets, or managing calendar events).

3. Limited Use Disclosure

The use and transfer of information received from Google APIs to any other app will adhere to Google API Services User Data Policy, including the Limited Use requirements.

4. Data Storage and Security

All data processed through this instance is kept within the private server environment and is not shared with any third parties. We implement industry-standard security measures, including SSL/TLS encryption (HTTPS), to protect the transmission of data. Data is stored locally in the instance's database and is not accessible to external entities.

5. Data Sharing

We do not share, sell, or distribute any user data or account information with third-party developers, advertisers, or any other entities. Everything happens within this private, self-hosted infrastructure.

6. Changes to This Policy

This policy may be updated periodically to reflect changes in our practices or for legal reasons. Any significant changes will be reflected in the "Last Updated" date at the top.
