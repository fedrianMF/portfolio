Política de Privacidad

Última actualización: 2 de enero de 2026

Esta política de privacidad describe cómo se gestionan los datos en la instancia privada de n8n alojada en n8n.fedrian.dev.

1. Uso de los Datos

Esta aplicación es de uso estrictamente personal y privado. Se utiliza exclusivamente para la automatización de flujos de trabajo del propietario del dominio.

2. Acceso a Datos de Google

Cuando el usuario autoriza el acceso a servicios de Google (OAuth), los datos obtenidos se utilizan únicamente para ejecutar las automatizaciones programadas (ej. leer correos, actualizar hojas de cálculo). No se almacenan datos fuera de la base de datos local del servidor.

3. Intercambio de Datos

No compartimos, vendemos ni distribuimos ninguna información personal o de cuenta con terceros. Todo el procesamiento de datos ocurre dentro de este servidor privado en DigitalOcean.

4. Seguridad

Implementamos medidas de seguridad estándar, incluyendo cifrado SSL (HTTPS) para proteger la transmisión de datos.
